
## 技术栈

**前端**
+ vue
+ element-ul
+ axios

**后端**
+ SpringBoot
+ SpringMVC
+ MybatisPlus

**数据库**
+ Mysql (主从分离)

**中间件**
+ Redis (热数据缓存)
+ Nginx (反向代理+静态资源部署)

**第三方服务**
+ 腾讯云短信服务

**AOP**
+ 全局异常处理器
+ 统一封装结果类
+ 过滤器
